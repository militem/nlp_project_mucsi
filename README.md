# Buscador de Hoteles mediante Lenguaje Natural

Este proyecto consiste en el desarrollo de un buscador de hoteles avanzado que utiliza técnicas de **Procesamiento de Lenguaje Natural (NLP)** y **Minería de Texto**. 

El objetivo principal es superar las limitaciones de los buscadores tradicionales basados únicamente en palabras clave, permitiendo una interacción más natural y semántica con la información.

## Características Principales

*   **Análisis Semántico**: Capacidad para comprender la intención del usuario y procesar consultas en lenguaje natural.
*   **Extracción de Entidades**: Identificación automática de elementos clave como ubicación, servicios y descripciones de alojamientos.
*   **Procesamiento de Datos**: Extracción y limpieza de información proveniente de diversas fuentes web para construir una base de conocimientos sólida.
*   **Relaciones Semánticas**: Análisis de las relaciones entre diferentes entidades para ofrecer resultados más precisos y relevantes.
*   **Sistema de Recomendación basado en Grafos**: Toda la información se estructura en un grafo de conocimiento para capturar relaciones complejas entre hoteles, servicios y ubicaciones.
*   **GraphRAG**: Implementación de Retrieval-Augmented Generation sobre grafos para proporcionar respuestas contextuales y precisas basadas en la estructura de red del conocimiento.

## Tecnologías Utilizadas

*   **Python**: Lenguaje principal para el desarrollo de scripts de extracción y procesamiento.
*   **Pandas & JSON**: Gestión eficiente de grandes volúmenes de datos de alojamientos.
*   **NLP & Text Mining**: Técnicas avanzadas para el análisis de texto y semántica.
*   **Neo4j**: Base de datos de grafos para almacenar y consultar el conocimiento.
